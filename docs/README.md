# Documentation Structure

This project follows the [Diátaxis framework](https://diataxis.fr/) for documentation organization:

- **[Tutorials](tutorials/)**: Learning-oriented. "How to get started."
- **[Guides](guides/)**: Task-oriented. "How to do X."
- **[Reference](reference/)**: Information-oriented. API docs, configurations.
- **[Explanation](explanation/)**: Understanding-oriented. Background context, architecture decisions.
